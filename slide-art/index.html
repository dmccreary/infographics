    Slide Art Inforgraphics

    Here is a detailed list of slide art types available in presentation tools such as Google Slides or MicroSoft PowerPoint**, organized by category.

    These categories group diagrams by their purpose, and each type has multiple layouts. While the exact options may vary slightly by PowerPoint version, these are the core SmartArt graphic types included in most recent versions of Microsoft 365, PowerPoint 2016â€“2021, and Office 2021.

    Categories of SmartArt in PowerPoint

    Each category contains layouts optimized for different purposes:

    List Diagram Types

    List infographics are to present information that does not need to follow a specific order. Items in these list can be rearranged without changing the meaning of the infographic.

    1. Basic Block List
    2. Vertical Bullet List
    3. Horizontal Bullet List
    4. Picture Accent List
    5. Vertical Picture List
    6. Grouped List
    7. Bending Picture Accent List
    8. Circle Picture List
    9. Continuous Picture List
    10. Grid List
    11. Horizontal Picture List
    12. Vertical Box List
    13. Vertical Chevron List
    14. Vertical Block List
    15. Staggered Process List

    Process Diagram Types

    Process diagrams are used to show progression, steps in a task, or a timeline. In these infographic types, order is key and a rearrangement might change the mean or make the diagram incorrect. These basic diagrams do not have loops or feedback.

    1. Basic Process
    2. Accent Process
    3. Process Arrows
    4. Continuous Block Process
    5. Chevron Process
    6. Circular Bending Process
    7. Picture Accent Process
    8. Step Up Process
    9. Upward Arrow Process
    10. Downward Arrow Process
    11. Vertical Process
    12. Text Cycle
    13. Gear Process
    14. Timeline
    15. Segmented Process

    Cycle

    Used to show repeating processes*1. or circular flows that include feedback. Many of these cycle types include relationships that are labeled.

    1. Basic Cycle
    2. Continuous Cycle
    3. Cycle Matrix
    4. Circle Arrow Process
    5. Text Cycle
    6. Block Cycle
    7. Radial Cycle
    8. Gear Cycle
    9. Picture Accent Cycle
    10. Segmented Cycle
    11. Nested Cycle
    12. Funnel Cycle

    Hierarchy Diagrams

    Used to show organizational structure*1. or family trees**.

    1. Organization Chart
    2. Name and Title Organization Chart
    3. Picture Organization Chart
    4. Horizontal Organization Chart
    5. Table Hierarchy
    6. Labeled Hierarchy
    7. Horizontal Labeled Hierarchy
    8. Circle Picture Hierarchy
    9. Horizontal Multi-Level Hierarchy
    10. Hierarchy List

    Relationship Diagrams

    Used to show interconnections*1. or how items relate to one another**.

    1. Basic Venn
    2. Radial Venn
    3. Target Diagram
    4. Basic Radial
    5. Circular Radial
    6. Counterbalance Arrows
    7. Balanced Matrix
    8. Segmented Pyramid
    9. Converging Radial
    10. Diverging Radial
    11. Opposing Ideas
    12. Interconnected Rings
    13. Hexagon Cluster
    14. Plus and Minus

    Grid or Matrix Diagrams

    Used to show relationships of components to a whole based on two dimensions.

    1. Basic Matrix
    2. Grid Matrix
    3. Titled Matrix
    4. Labeled Matrix

    Pyramid Diagrams

    Used to show proportional, hierarchical relationships.

    1. Basic Pyramid
    2. Inverted Pyramid
    3. Pyramid List
    4. Segmented Pyramid
    5. Picture Pyramid
    6. Funnel
    7. Pyramid Accent List

    Picture Digrams

    Used to incorporate images into lists, cycles, or relationships.

    1. Picture Accent List
    2. Picture Accent Process
    3. Picture Accent Cycle
    4. Vertical Picture List
    5. Horizontal Picture List
    6. Bending Picture Accent List
    7. Circle Picture Hierarchy
    8. Picture Caption List
    9. Picture Grid
    10. Picture Stack

    Additional Paramters to Diagrams

    1. Customized Layouts: Shapes, colors, styles, and text formatting can all be adjusted within an infographic.
    2. Animations: can be applied to individual components for presentations. For example, circles can move along lines to show direction of flow of data or causality.
