    Mascot: Percy the Peacock

    Character Profile

    • Name: Percy the Peacock
    • Species: Peacock
    • Personality: Creative, proud (in a good way), encouraging, expressive
    • Catchphrases:
      • "Let's make it visual!"
      • "Display it with style!"
      • "Let's spread some knowledge!"
    • Visual: Royal blue body, orange-gold tail feather accents, small round designer glasses, stylus tucked behind one wing, data-viz motifs in tail feather eye-spots
    • Art style: Modern flat vector, clean lines, transparent background

    Voice Characteristics

    • Uses clear, design-oriented language
    • Occasionally uses visual metaphors and display puns
    • Refers to students as "designers" or "visual thinkers"
    • Signature phrases: "Let's make it visual!", "Display it with style!", "Let's spread some knowledge!"

    Placement Rules

    Context Admonition Type Frequency
    General note / sidebar mascot-neutral As needed
    Chapter opening mascot-welcome Every chapter
    Key concept mascot-thinking 2-3 per chapter
    Helpful tip mascot-tip As needed
    Common mistake mascot-warning As needed
    Section completion mascot-celebration End of major sections
    Difficult content mascot-encourage Where students may struggle

    Do's and Don'ts

    Do:

    • Use Percy to introduce new topics warmly
    • Include a catchphrase in welcome admonitions
    • Keep dialogue brief (1-3 sentences)
    • Match the pose/image to the content type

    Don't:

    • Use Percy more than 5-6 times per chapter
    • Put mascot admonitions back-to-back
    • Use the mascot for purely decorative purposes
    • Change Percy's personality or speech patterns

    Image Specifications

    • Format: PNG with transparent background
    • Generation size: 1024x1024 pixels
    • Display size: 200x200 to 400x400 pixels
    • File size target: Under 100KB per image

    Next Steps

    1. Copy each pose prompt above into your AI image generator (ChatGPT/DALL-E, Midjourney, etc.)
    2. Generate at 1024x1024, save to docs/img/mascot/ with the filenames listed above
    3. Trim excess padding from images (AI generators often add too much whitespace)
    4. Set up CSS admonitions (Option 2 recommended) — see book-installer skill
    5. Add Percy character guidelines to CLAUDE.md for consistent AI-generated content
