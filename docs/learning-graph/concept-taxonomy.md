# Concept Taxonomy

This taxonomy organizes the 350 concepts into 14 categories for the Interactive Infographics for Intelligent Textbooks learning graph.

## Categories

### FOUND — Foundation Concepts
Core definitions and foundational ideas that underpin the entire course. Includes infographic types, intelligent textbooks, MicroSims, and basic interaction concepts.

### TAXON — Infographic Taxonomy
Classification systems for organizing infographic types by purpose, format, complexity, medium, and audience.

### SLIDE — Slide Art Diagrams
Specific diagram types from presentation tools including list, process, cycle, hierarchy, relationship, pyramid, matrix, and picture diagram types.

### FRAME — Visual Frameworks
Visual problem-solving frameworks including Dan Roam's Back of the Napkin, the Six W's, and the SQVID decision framework.

### CAUSE — Causal Loop Diagrams
Systems thinking concepts including causal loop diagrams, feedback loops, polarity, reinforcing/balancing loops, and leverage points.

### OVRLY — Overlayment Patterns
The four interactive overlay patterns (rectangular, polygon, callout-to-edge, floating label) and their editing, configuration, and JSON file formats.

### JSLIB — JavaScript Libraries
JavaScript visualization libraries (p5.js, D3.js, Chart.js, vis-network, Leaflet) and their specific features, functions, and chart types.

### WEBFD — Web Fundamentals
Core web technologies including HTML, CSS, JavaScript fundamentals, JSON, SVG, Canvas, DOM manipulation, and responsive design.

### MSIM — MicroSim Standards
MicroSim packaging conventions, directory structure, metadata, quality scoring, iframe embedding, and standardization checklist.

### GENAI — Generative AI
AI-assisted infographic creation including text-to-image generation, overlay generation, code generation, prompt engineering, and quality review.

### TRACK — Tracking and Analytics
Student interaction tracking, xAPI, event logging, engagement metrics, learning analytics, and instructor dashboards.

### LEARN — Learning Science
Educational theory including cognitive load, dual coding, multimedia learning, Bloom's Taxonomy, scaffolding, and assessment.

### DEPLY — Deployment and Tools
MkDocs, GitHub Pages, Git version control, CI/CD, Markdown, site configuration, and deployment workflows.

### ADVIZ — Advanced Visualization
Advanced diagram types, interaction patterns, design patterns, accessibility, and specialized visualization techniques.
