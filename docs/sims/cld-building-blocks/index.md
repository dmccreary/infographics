## About This MicroSim

This step-through explorer introduces the three fundamental building blocks of a Causal Loop Diagram (CLD): **variables**, **causal links**, and **polarity indicators**. Each stage adds one building block to a simple example, showing how they combine to form a complete feedback loop.

### The Three Building Blocks

1. **Variables** — Quantities that can increase or decrease over time, shown as labeled ovals
2. **Causal Links** — Arrows connecting variables to show cause-and-effect relationships
3. **Polarity Indicators** — Plus (+) or minus (−) signs on arrows showing whether variables change in the same or opposite directions

## Lesson Plan

### Learning Objective

Explain the three building blocks of a causal loop diagram by interacting with each element and observing how they combine.

### Activities

1. Start at Stage 1. Identify the three variables and think about how they might relate.
2. Advance to Stage 2. Trace the arrows — which variable influences which?
3. Advance to Stage 3. Notice the + signs. What does it mean that all links are positive?
4. Hover over each element for additional details about its role in the loop.
